# Claude Demo Skill

## Overview
This is a demonstration skill for Claude that showcases its abilities in code generation, explanation, and problem solving.

## Features
1. **Code Generation**: Generate code snippets in multiple programming languages
2. **Code Explanation**: Explain how code works and its logic
3. **Problem Solving**: Help solve coding problems step by step
4. **Code Review**: Review code quality and suggest improvements

## Usage Examples

### Example 1: Code Generation
```
Generate a Python function that calculates the factorial of a number
```

### Example 2: Code Explanation
```
Explain this JavaScript code:
function fibonacci(n) {
    if (n <= 1) return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}
```

### Example 3: Problem Solving
```
How would you solve the two sum problem in Python?
```
